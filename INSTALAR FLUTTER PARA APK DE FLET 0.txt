INSTALAR FLUTTER PARA APK DE FLET 0.84.0 EN WINDOWS

Lo que ya tenemos:

Python 3.13 con venv
Flet 0.84.0 instalado
App mobile en mobile/app.py (Flet)
Java JDK (Eclipse Temurin) — descargado, pendiente instalar
Flutter 3.22.3 ZIP — descargado

Lo que falta hacer:

Instalar Java JDK (el .msi que descargaste)
Extraer Flutter 3.22.3 ZIP a C:\flutter
Agregar C:\flutter\bin al PATH de Windows
Instalar Android Studio + Android SDK
Ejecutar flutter doctor y corregir lo que falte
Ejecutar flet build apk en la carpeta del proyecto

El archivo a empaquetar:

Ruta: mobile/app.py
Repo: https://github.com/milenamondaca17-gif/juana-cash
Flet version: 0.84.0
Python: 3.13

Comando final para generar la APK:
powershellcd mobile
flet build apk --project "JuanaCash" --org "com.cammus25"
Notas importantes:

Flet 0.84.0 es compatible con Flutter 3.22.x
La APK se conecta al backend por IP (configurable en la app)
